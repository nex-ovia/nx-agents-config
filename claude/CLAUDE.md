# nx-agents-config

This Claude Code instance uses centralized resources from `~/.nx-agents-config/`.

## Shared Resources
- **Skills**: `~/.nx-agents-config/shared/skills/`
- **Rules**: `~/.nx-agents-config/shared/rules/`
- **Memory**: `~/.nx-agents-config/shared/memory/`

## Usage
Load relevant skills and follow rules as needed. Use the `~/.nx-agents-config/shared/memory/` directory for persisting context across sessions.
